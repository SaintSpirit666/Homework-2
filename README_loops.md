## Список заданий

### Задание 1: Вывод средних букв слова
Программа выводит среднюю букву (если количество нечетное) или две средние буквы (если четное).

**Пример работы:**
```python
word = 'test'
Результат: es

word = 'testing'
Результат: t
```

**Реализация:** Используется срезы строк и проверка четности через остаток от деления.

---

### Задание 2: Сумма чисел до первого нуля
Программа запрашивает числа у пользователя и суммирует их до первого введенного нуля.

**Пример работы:**
```
Введите число: 1
Введите число: 4
Введите число: 6
Введите число: 0
Результат: 11
```

**Реализация:** Цикл `while True` с условием выхода при вводе 0.

---

### Задание 3: Dating-сервис (сватовство)
Программа сортирует списки парней и девушек по алфавиту и создает пары с одинаковыми индексами. Если количество не совпадает, выводится предупреждение.

**Пример работы:**
```python
boys = ['Peter', 'Alex', 'John', 'Arthur', 'Richard']
girls = ['Kate', 'Liza', 'Kira', 'Emma', 'Trisha']

Результат:
Идеальные пары:
Alex и Emma
Arthur и Kate
John и Kira
Peter и Liza
Richard и Trisha
```

**Реализация:** Функция `sorted()` для сортировки и цикл `for` для вывода пар.

---

### Задание 4: Конвертация температуры из Фаренгейта в Цельсий
Программа рассчитывает среднюю температуру в Цельсиях для каждой страны из списка температур в Фаренгейтах.

**Формула конвертации:** `C = (F - 32) × 5/9`

**Пример работы:**
```python
countries_temperature = [
    ['Таиланд', [75.2, 77, 78.8, 73.4, 68, 75.2, 77]],
    ['Германия', [57.2, 55.4, 59, 59, 53.6]],
    ['Россия', [35.6, 37.4, 39.2, 41, 42.8, 39.2, 35.6]],
    ['Польша', [50, 50, 53.6, 57.2, 55.4, 55.4]]
]

Результат:
Средняя температура в странах:
Таиланд - 23.9 С
Германия - 13.8 С
Россия - 3.7 С
Польша - 12.0 С
```

**Реализация:** List comprehension для конвертации и цикл для обработки каждой страны.

---

### Задание 5: Валидация транспортных номеров (необязательное)
Программа проверяет российские автомобильные номера на валидность по формату: 1 буква, 3 цифры, 2 буквы, 2-3 цифры. Использует только допустимые буквы кириллицы.

**Допустимые буквы:** А, В, Е, К, М, Н, О, Р, С, Т, У, Х

**Пример работы:**
```python
car_ids = ['А222ВС96', 'АБ22ВВ193']

Результат:
Номер А222ВС валиден. Регион: 96
Номер АБ22ВВ193 не валиден
```

**Реализация:** Регулярные выражения (модуль `re`) для проверки формата.

---

### Задание 6: Анализ логов просмотров (необязательное)
Программа анализирует поток логов и вычисляет среднее количество просмотров на уникального пользователя.

**Формат данных:** `пользователь,дата;просмотры`

**Пример работы:**
```python
stream = [
    'user4,2021-01-01;3',
    'user3,2022-01-07;4',
    'user2,2022-03-29;1',
    'user1,2020-04-04;13',
    ...
]

Результат:
Среднее количество просмотров на уникального пользователя: 23.25
```

**Реализация:** 
- Регулярные выражения для парсинга логов
- Словарь для аккумуляции просмотров по пользователям
- Вычисление среднего значения
